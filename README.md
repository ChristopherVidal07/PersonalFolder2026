# Personal Portfolio Website

**Author:** Christopher Vidal

## Project Overview

This project is a personal portfolio website created from scratch using
HTML, CSS, and JavaScript. The website represents my professional
identity and showcases my skills, education, experience, and software
development projects. It is designed as a modern single-page portfolio
with smooth navigation and interactive elements.

The portfolio is intended to help employers and recruiters learn more
about my technical skills, projects, and contact information.

# Part 2: Design

-   Style: Minimalist and Professional
-   Color Scheme: Blue, White, and Navy
-   Font: Bricolage Grotesque
-   Homepage Layout: Navigation, Hero, About, Experience, Skills,
    Projects, Contact, Footer
-   Projects displayed using cards.
-   Responsive layout using Flexbox and media queries.
-   Accessibility through readable fonts, contrast, semantic HTML, and
    alt text.
-   Images, illustrations, and icons will be used to create a modern
    appearance.

------------------------------------------------------------------------

# Part 3: Interactivity

Interactive features include:

-   Navigation links
-   Hover effects
-   Contact button
-   Email link
-   Project carousel using JavaScript
-   Previous and Next buttons

The contact button uses a mailto link.

JavaScript changes project images and text in the carousel without
reloading the page.
